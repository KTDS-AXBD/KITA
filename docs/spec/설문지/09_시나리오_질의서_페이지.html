분석 레벨 정의 (L0 ~ L5)

시나리오별 표·문항에서 동일한 용어로 쓰입니다. 옵션 선택 시 참고하세요.
L0
단순 조회·통계 · 기존 ChatGIVC 챗봇 / 단일 화면 통계 (추가 사업 이전 수준) — 비교 baseline
L1
정적 지식그래프 · 한 화면 위에 품목·기업·HS 관계가 펼쳐진 정적 그래프 1장
L2
시나리오 Q&A 시연 · 1~2건의 질문에 대해 그래프 + 표 + 차트 조합 응답
L3
다단계 추론 + 근거 추적 · "왜 이 결과인가" 노드별 출처·신뢰도 hover, 다단계 follow-up 가능
L4
가중치 조정 What-If · 슬라이더로 가중치 변경 시 결과 실시간 재계산, 반대 추천(제외 사유) 포함
L5
정책 옵션 자동 생성 · 정책 카드 자동 제시 + 실시간 데이터·알림 결합 (본 사업화 v1 목표)

시나리오별 데이터 매핑 (S1 ~ S7)

현재 = 외부 공개 + ChatGIVC 1차 자료로 도달 가능 · +추가 = 핵심 추가 데이터 확보 시 · 최대 = 본 사업화 v1 목표
ID 시나리오 · 핵심 질문 현재 가능 추가 확보 시 가능
S1 나프타 공급망 영향도
가격·공급 변동 시 전후방 산업·기업 영향		 L1 HS4 (2710) 무역 통계 · 수입 상위국 · 위험 시그널(hs_anomaly)까지 시각화. 종류별·전후방은 가상값 L3 경질·중질 종류별 + 품목→품목 전후방 매핑 → 후방 영향 실데이터 추론. L5 + 정책 옵션
S2 호르무즈 해협 지정학 리스크
지정학 이벤트 시 영향 품목·대체 공급선		 L0~L1 HS·산업 매핑은 있으나 지정학 데이터·이벤트→품목 연관 매트릭스 부재. 외부 뉴스 워드클라우드 수준 L2 지정학 뉴스 매핑 + 이벤트→품목 연관 정의. L3 + KOTRA 현지 정보 결합
S3 리튬·니켈 원자재 가격 폭등
2차전지·전기차·로봇 산업 전파 경로		 L1 원자재 HS 통계 + 2차전지·전기차 산업 매핑 (산업 단위). 가격 변동 추적 불가 L3 실시간 가격 변동 트래커 + 전파 경로 매트릭스 → 다단계 추론
S4 소부장 자립화 R&D 적합 기업 추천 🥇 MAIN
산기평 공고 시 기술·실적 기반 자립화 후보 기업 자동 추천		 L2 기업 마스터·재무·R&D 지표·특허 모두 보유. 산기평 공고만 가상값으로 시연 가능 (Top 5 + 근거 그래프 · 데모 사이트 현 수준) L4 산기평 공고 실데이터 + 특허청 트렌드 → 가중치 What-If · 반대 추천. L5 + 정책 옵션
S5 EWS 정책 대응안
조기 경보 발생 시 정책 옵션 매트릭스		 L1 hs_anomaly(위험 알림) 보유. 정책 카드 객체 자체가 없음 L3 정책 카드 라이브러리 + EWS 팀 협업 룰. L5 + 자동 정책 생성
S6 공작기계 가치사슬 가시화 🥈 SUB
감속기·CNC 등 핵심 부품의 소재→부품→장비 가치사슬을 한 화면		 L2 공작기계 품목군 GIVC 보유 데이터로 시연. 일본·중국 수입 비중·핵심 기업·공정 단계 실데이터 (데모 사이트 현 수준) L3 전후방 1~2단계 매핑 + 실시간 뉴스 매칭 → 후방 영향 추론까지 확장
S7 한일 무역협정 영향 (가상)
협정 변경 시 영향 품목·기업		 L0 협정 텍스트·외교 데이터 부재. 거의 가상값에 의존 L1~L2 협정 텍스트 + 외교 데이터 확보 시 부분 시연. 우선순위 낮음

▸ 5/21 합의 흐름 충실 옵션 = 옵션 C (S1 + S2 호르무즈 결합) · 데이터 가상 비중 크지만 시의성 큼
