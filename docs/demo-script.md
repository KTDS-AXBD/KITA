# KITA PoC 시연 스크립트 (F011)

> Prototype 리뷰 시연용 **핵심 흐름 outline** (25~35분). 대상: 산자부/산업부 의사결정자. 발표자: 서민원 책임.
> 핵심 목표: "GIVC 위 온톨로지" 가치를 화면으로 보여주고 **고객이 자발적으로 보유 데이터를 꺼내게** 유도.
> 사전: 라이브 URL 가용 확인(`bash scripts/healthcheck.sh <URL>`) + 오프라인 백업 대기(`pnpm serve:offline`).

---

## 흐름 한눈에

```
① Landing (2~3분)  →  ② S4 R&D 추천 (10~14분, 메인)  →  ③ S6 톨루엔 (6~8분)  →  ④ About (3~5분)  →  ⑤ 마무리·데이터 유도 (3~5분)
```

데이터 표기 강조: 모든 화면에서 **⭐실 / △추정 / ※가상**을 짚어 신뢰성 선점.

---

## ① Landing — 데모의 위치 (2~3분)

- **화면**: `/` (Landing)
- **조작**: 히어로 → 시나리오 카드 2개 소개
- **핵심 메시지**:
  - "설문 대신 데모로 — 현재 GIVC 보유 데이터만으로 가능한 수준 + 추가 데이터 결합 시 확장 path를 한 화면에."
  - ProvenanceLegend(⭐△※) 가리키며 "모든 수치에 출처를 답니다. 가상은 ※로 명시 — 신뢰성 오인 없습니다."
- **전환**: "메인 시연 S4부터 보시죠" → S4 카드 클릭

## ② S4 — R&D 적합 기업 추천 (10~14분, 메인)

- **화면**: `/scenario/rnd`
- **조작 순서**:
  1. **Top 5 표** 제시 — "산기평 공고가 뜨면 적합 기업 Top5 + 근거 + 반대 추천이 자동 생성됩니다."
  2. **가중치 슬라이더** 조작(특허 0.30→0.15) — "가중치를 바꾸면 순위가 **실시간 재계산**(<100ms)됩니다." (순위 변동 시연)
  3. **선정근거 지식그래프** — 행 hover → 근거 노드 하이라이트, 노드 hover → 출처 tooltip
  4. **Data Expansion Hints** 토글 "산기평 공고 실데이터" ON — "정확도가 **65% → 88%**로. 이게 데이터 한 종을 더 얹었을 때의 효과입니다."
  5. (선택) **What-If 토글** ON → "특허 가중치를 낮추면?" 질의 — "실 LLM이 GIVC 컨텍스트로 답합니다. 시범 기능이며 세션당 3회 제한." (실패 시 정적 응답 자동 대체 — 당황 금지)
- **데이터 유도 포인트** 🎯:
  - C1: "여기 특허 수는 ※가상입니다 — **특허청 데이터**가 있으면 실값으로 바뀝니다." → *고객이 특허 데이터 보유 여부 언급하도록*
  - C2: "산기평 공고 텍스트가 있으면 매칭 정확도가 올라갑니다." → *공고 데이터 접근성 유도*

## ③ S6 — 톨루엔 단일 품목 (6~8분)

- **화면**: `/scenario/toluene`
- **조작**:
  1. **중앙 지식그래프** — "톨루엔 하나로 HS코드·수입국·핵심기업·전후방을 한 화면에. '창 띄웠다 끄는' 페인을 통합으로 해결."
  2. **수출입 추이 차트** + 이상치 마커 — "이동평균 ±2σ 초과 이상 시그널 자동 표시."
  3. **핵심·예비 기업표** (hover → 그래프 연동)
  4. **워드클라우드** — "산업 단위 뉴스. 품목 단위로 내려오면 위험 알림이 더 깊어집니다."
- **데이터 유도 포인트** 🎯:
  - C3: "전후방 노드는 ※가상입니다 — **전후방 매핑 데이터**가 있으면 영향 분석이 자동화됩니다." → *전후방 데이터 유도*
  - C4: "뉴스를 품목 단위로 매칭하려면 실시간 뉴스 소스가 필요합니다." → *뉴스 데이터 유도*

## ④ About — 데이터 출처 (3~5분)

- **화면**: `/about/data` (+ `/about/ontology`)
- **핵심 메시지**:
  - 데이터 카탈로그 표 — "⭐실 7 / △추정 1 / ※가상 4. 실데이터 58%. 시연 단계에서 솔직하게 구분합니다."
  - "본 사업화 시: Mock → GIVC API 연동, 가짜 LLM → 사내 LLM+RAG. 표기 규칙(⭐△※)은 유지."
- **데이터 유도 포인트** 🎯:
  - C5: "지금 ※가상인 항목들이 곧 귀측이 보유하신 데이터입니다 — 어떤 걸 연동할 수 있을까요?" → *직접 질의*
