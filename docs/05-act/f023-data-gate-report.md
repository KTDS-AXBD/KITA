## 4. 판정 근거

- **외부 신청 0**: 관세청 키(`DATA_GO_KR_KEY`)·DART 키(`OPENDART_KEY`) 기존 보유. 기계 HS는 hsSgn 파라미터 교체만으로 조회(F019 실측 재확인). corpCode.xml은 키 1개로 다운로드.
- **F023 DoD 충족 가능**: 관세청 기계 HS + DART 기계사 실매출 → 정식 적재(검증 PASS) + Repository 교체로 화면 코드 변경 0 입증 경로 확보.
- **교훈 재확인**: corpCode.xml 압축은 `application/x-msdownload`(ZIP)로 내려옴 → `unzip` 부재 환경은 python3 `zipfile`로 해제. `<corp_name>`과 `<stock_code>` 사이 `<corp_eng_name>` 태그 있음 → 정규식보다 XML 파서 권장.
